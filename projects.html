Projects

This page lists the major open source projects I have worked on, organized by area.

Desktop & Linux Platform

Project Description Technologies
GNOME / Telepathy Contributed to the GNOME desktop communication stack, including work on the Telepathy Framework and early WebRTC implementation on WebKit. This work enabled real-time communication features in the GNOME desktop environment. C, GLib, Telepathy, WebRTC, WebKit
KDE Plasma / KBFX Worked on KDE Plasma desktop technologies. Developed KBFX, a customizable application menu system for KDE that provided an alternative to the standard K Menu with theming and layout flexibility. C++, Qt, KDE Plasma, CMake
PlexyDesk Developed the PlexyDesk Compositor Protocol and Wayland bridge. PlexyDesk is a desktop extension framework that provides compositor-level desktop widgets and a protocol for desktop shell integration with Wayland. C++, Qt, Wayland, OpenGL, EGL
XQuadro Developed XQuadro, an X11 display server bridge that enabled multi-display and display server bridging capabilities for X11-based systems. C, X11, Xlib, XCB

Browser Engines

Project Description Technologies
V8 JavaScript Engine Contributed to the V8 JavaScript engine with a focus on speed and performance optimizations. Worked on JIT compiler improvements and runtime performance tuning. C++, JavaScript, JIT, V8
SpiderMonkey Worked on Mozilla's SpiderMonkey JavaScript engine, implementing GObject Introspection support. This enabled automatic language bindings for GNOME-based JavaScript applications. C++, JavaScript, GObject Introspection, GLib
Mozilla / Gecko Contributed to the Mozilla Gecko rendering engine, working on CSS animations support and rendering pipeline improvements. C++, CSS, XUL, Gecko
WebKit Contributed to WebKit with work on HTML5 video support and WebRTC integration. This work was part of enabling real-time communication and media playback in WebKit-based browsers. C++, WebKit, HTML5, WebRTC, GStreamer
Google Chrome (E-Ink Rendering) Worked on Google Chrome with e-ink rendering support, enabling the browser to work effectively on e-ink display devices. C++, Chromium, Skia, E-Ink

Mobile & Multimedia

Project Description Technologies
GStreamer Contributed to GStreamer, the open source multimedia framework. Worked on pipeline elements, codec integration, and multimedia playback for embedded and mobile platforms. C, GStreamer, Glib, multimedia codecs
Nokia Browser (Mozilla-based) Worked on the Mozilla-based Nokia Browser for Nokia's mobile platforms. Contributed to rendering, performance, and platform integration for mobile browsing. C++, Mozilla, Gecko, Maemo, Harmattan
Nokia Harmattan / MeeGo Contributed to the Nokia Harmattan platform and MeeGo mobile operating system. Worked on system-level components, UI framework integration, and application platform support. C++, Qt, MeeGo, Harmattan, Linux

Blockchain & Web3

Project Description Technologies
Web3 Systems Developed Web3 infrastructure and tooling, working on blockchain integration systems and decentralized application platforms. Solidity, JavaScript, Web3.js, Ethereum
BTC / Bitcoin Core Contributed to Bitcoin-based blockchain technologies, working on ledger systems and BTC protocol-level work. C++, Bitcoin, blockchain, cryptography
LiquidBTC Worked on LiquidBTC and related blockchain ledger systems, focusing on sidechain technology and federated blockchain infrastructure. C++, Elements, Liquid, sidechains

AI & Platforms

Project Description Technologies
CloudGPU Founded CloudGPU, an AI infrastructure and agent tooling platform. Provides GPU compute resources and tooling for AI agent development and deployment. Python, Go, Kubernetes, GPU, LLMs
Hola Agent Developed Hola Agent, an AI agent framework for building autonomous agents with tool-use capabilities, memory, and task planning. Python, LLMs, agent frameworks, tools
efinder.lk Created efinder.lk, an agentic fact-checking engine that uses AI agents to verify claims and provide sourced fact-checking. Python, LLMs, NLP, web scraping
Sharepanda.org Developed Sharepanda.org, a creator platform for content sharing and creator monetization. JavaScript, Node.js, Web3, creator tools
