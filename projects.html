Projects

Here are the main open source projects I've worked on, grouped by area.

Desktop & Linux Platform

Project Description Technologies
GNOME / Telepathy Worked on the GNOME desktop communication stack, including the Telepathy Framework and early WebRTC on WebKit. This brought real-time communication features to the GNOME desktop. C, GLib, Telepathy, WebRTC, WebKit
KDE Plasma / KBFX Worked on KDE Plasma desktop tech. Built KBFX, a customizable application menu for KDE that gave users an alternative to the standard K Menu with theming and layout options. C++, Qt, KDE Plasma, CMake
PlexyDesk Built the PlexyDesk compositor protocol and Wayland bridge. PlexyDesk is a desktop extension framework that gives you compositor-level widgets and a protocol for integrating desktop shells with Wayland. C++, Qt, Wayland, OpenGL, EGL
XQuadro Built XQuadro, an X11 display server bridge for multi-display and display server bridging on X11-based systems. C, X11, Xlib, XCB

Browser Engines

Project Description Technologies
V8 JavaScript Engine Contributed to the V8 JavaScript engine, focused on speed and performance. Worked on JIT compiler improvements and runtime performance tuning. C++, JavaScript, JIT, V8
SpiderMonkey Worked on Mozilla's SpiderMonkey JavaScript engine, adding GObject Introspection support. This made it possible to automatically generate language bindings for GNOME-based JavaScript apps. C++, JavaScript, GObject Introspection, GLib
Mozilla / Gecko Contributed to the Mozilla Gecko rendering engine, working on CSS animations and rendering pipeline improvements. C++, CSS, XUL, Gecko
WebKit Contributed to WebKit, working on HTML5 video support and WebRTC integration. This helped bring real-time communication and media playback to WebKit-based browsers. C++, WebKit, HTML5, WebRTC, GStreamer
Google Chrome (E-Ink Rendering) Worked on Google Chrome, adding e-ink rendering support so the browser works well on e-ink display devices. C++, Chromium, Skia, E-Ink

Mobile & Multimedia

Project Description Technologies
GStreamer Contributed to GStreamer, the open source multimedia framework. Worked on pipeline elements, codec integration, and multimedia playback for embedded and mobile platforms. C, GStreamer, Glib, multimedia codecs
Nokia Browser (Mozilla-based) Worked on the Mozilla-based Nokia Browser for Nokia's mobile platforms. Did rendering, performance, and platform integration work for mobile browsing. C++, Mozilla, Gecko, Maemo, Harmattan
Nokia Harmattan / MeeGo Contributed to the Nokia Harmattan platform and MeeGo mobile OS. Worked on system-level components, UI framework integration, and application platform support. C++, Qt, MeeGo, Harmattan, Linux

Blockchain & Web3

Project Description Technologies
Web3 Systems Built Web3 infrastructure and tooling, working on blockchain integration systems and decentralized application platforms. Solidity, JavaScript, Web3.js, Ethereum
BTC / Bitcoin Core Contributed to Bitcoin-based blockchain technologies, working on ledger systems and protocol-level BTC work. C++, Bitcoin, blockchain, cryptography
LiquidBTC Worked on LiquidBTC and related ledger systems, focusing on sidechain technology and federated blockchain infrastructure. C++, Elements, Liquid, sidechains

AI & Platforms

Project Description Technologies
CloudGPU Founded CloudGPU, a platform for AI compute and agent tooling. Provides GPU resources and tooling for building and deploying AI agents. Python, Go, Kubernetes, GPU, LLMs
Hola Agent Built Hola Agent, a framework for creating autonomous AI agents with tool use, memory, and task planning. Python, LLMs, agent frameworks, tools
efinder.lk Created efinder.lk, a fact-checking engine that uses AI agents to verify claims and give you sourced fact-checks. Python, LLMs, NLP, web scraping
Sharepanda.org Built Sharepanda.org, a creator platform for content sharing and monetization. JavaScript, Node.js, Web3, creator tools
