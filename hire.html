Hire Me

I'm available for software consulting and system architecture work. I've been doing this for over a decade across open source desktop, browser engines, mobile, blockchain, and AI infrastructure. If your team needs to design, build, or ship something complex, I can help.

Software Consultancy

I do hands-on consulting for teams that need deep technical expertise in any of these areas:

  • Browser engine internals — V8, SpiderMonkey, WebKit, Mozilla/Gecko performance work, rendering pipeline optimization, and JavaScript engine tuning.
  • Linux desktop & platform — GNOME, KDE, Wayland compositor protocols, display server bridges, desktop integration.
  • Mobile & multimedia — GStreamer pipelines, mobile OS platforms, multimedia stack architecture.
  • Blockchain & Web3 — ledger system design, BTC-based protocols, decentralized application architecture.
  • AI infrastructure — agent tooling, GPU orchestration, inference platform design, agentic system architecture.
  • Code review & mentoring — reviewing critical codepaths, improving code quality, helping your team level up.

Architecture Design

I help teams design scalable, maintainable system architectures from scratch, or refactor existing ones. Here's what that looks like:

Service Description
Greenfield Architecture End-to-end system design for new products, from component breakdown and technology selection to data flow, deployment topology, and scaling strategy.
Architecture Review Audit of existing systems to find bottlenecks, technical debt, and failure modes. You get a written report with prioritized recommendations.
Performance Engineering Profiling and optimization of latency-sensitive systems: rendering pipelines, inference serving, network I/O, and concurrency bottlenecks.
Platform Design Design of internal developer platforms, CI/CD pipelines, release infrastructure, observability stacks.
Distributed Systems Design of distributed ledgers, consensus mechanisms, replication strategies, fault-tolerant service architectures.

Engagement Model

I work on a project or retainer basis, remote or on-site. Typical engagements:

  • Short-term advisory — 1 to 4 weeks of focused consulting, architecture review, or technical due diligence.
  • Project engagements — 1 to 6 months of embedded work designing and building a specific system or subsystem.
  • Retainer arrangements — ongoing advisory availability for teams that need a trusted architect on call.

Every engagement starts with a free initial call to understand what you need and figure out if I'm the right fit. I'll send you a written scope, timeline, and estimate before any work begins.
