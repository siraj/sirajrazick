About Me

I'm Siraj Razick, an open source developer. I've spent years working across a pretty wide range of technologies: Linux desktop, browser engines, mobile OS work, multimedia, blockchain, and more recently AI infrastructure.

What I Do

A quick rundown of the areas I work in:

  • Linux desktop (GNOME, KDE, Wayland, compositor protocols)
  • Browser engines (V8, SpiderMonkey, WebKit, Mozilla/Gecko)
  • Mobile and multimedia (GStreamer, Nokia Harmattan, MeeGo)
  • Blockchain and Web3 (Bitcoin, LiquidBTC, ledger systems)
  • AI infrastructure (CloudGPU, Hola Agent)
  • Creator platforms (Sharepanda.org, efinder.lk)

Background

I've been in open source for a long time now. I started out on the GNOME communication stack, working on the Telepathy Framework and some of the early WebRTC stuff on WebKit. From there I got into browser engine internals: V8 performance work, SpiderMonkey improvements, CSS animations in Mozilla/Gecko.

On the desktop side, I worked on KDE Plasma, built the KBFX application menu, and developed the PlexyDesk compositor protocol with a Wayland bridge. I also built XQuadro, an X11 display server bridge.

In the mobile world, I contributed to GStreamer, the Mozilla-based Nokia Browser, and Nokia Harmattan / MeeGo.

Lately I've been spending more time on blockchain (Web3, BTC, LiquidBTC, ledger systems) and AI infrastructure (CloudGPU, Hola Agent, efinder.lk, Sharepanda.org).
