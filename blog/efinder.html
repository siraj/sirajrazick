efinder.lk: Agentic Fact-Checking with AI

2023-11-20

I recently released efinder.lk, an agentic fact-checking engine that uses AI agents to verify claims and provide sourced fact-checking.

The problem efinder.lk addresses is the spread of misinformation online. Traditional fact-checking is manual and slow. efinder.lk uses AI agents that can:

  • Parse and understand claims in natural language
  • Search for relevant sources and evidence
  • Cross-reference multiple sources
  • Provide a verdict with citations

The system is built using the Hola Agent framework and uses LLMs for natural language understanding. Agents are given tools for web search, source retrieval, and claim analysis. The agents work through a multi-step reasoning process to arrive at a verdict.

You can try it at efinder.lk.
