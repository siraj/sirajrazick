Announcing the First Public Source Code Release of PlexyDesk

2024-02-20

Today is a happy day for PlexyDesk.

After more than two years of work, I am releasing the first public version of the full PlexyDesk source code. This release represents a long journey of design, experimentation, rebuilding, debugging, and pushing through all the difficult work that happens behind the scenes before a project is ready to be shared openly.

PlexyDesk is a GPU-accelerated display server stack, desktop shell, compositor, and application toolkit for GNU/Linux and BSD. The goal is simple: to build a fast, modern, and flexible desktop platform that gives developers and users more control over the desktop experience.

Over the past two years, PlexyDesk has grown from an idea into a working platform. Recent advancements in AI helped accelerate parts of the development process, allowing me to move faster, test ideas more quickly, and improve the project with better focus.

As a by-product of this journey, I also ended up creating Hola Agent, an AI agent project that came out of the same development work.

This release is not just a code drop. It is the result of a lot of hard work, late nights, technical experiments, and persistence. I am excited to finally share it publicly and invite developers, contributors, and Linux desktop enthusiasts to explore the project.

What is included

The public release includes the PlexyDesk source code and related repositories under the PlexyDesk GitHub organization.

Explore the source code here:

https://github.com/orgs/plexydesk/repositories

The new PlexyDesk website is also live:

https://plexydesk.com

Developer documentation and project notes are available on the new developer wiki:

https://plexydesk.org

Why this matters

The Linux desktop still has room for new ideas.

PlexyDesk is my attempt to rethink parts of the desktop stack with a focus on performance, modern graphics, developer flexibility, and a more integrated shell experience.

This release opens the door for others to inspect the work, build on it, test it, report issues, and contribute.

The project is still evolving, but making the source code public is an important milestone. It means PlexyDesk can now grow in the open.

A note of thanks

To everyone who has followed the project, encouraged the work, tested builds, or simply believed that this was worth building: thank you. This is the beginning of a new public chapter for PlexyDesk as a complete display stack for GNU/Linux and FreeBSD.
