PlexyDesk Wayland Bridge Update

2023-05-02

I have updated the PlexyDesk Wayland bridge with improved compositor protocol support and better multi-monitor handling. This update has been in the works for a while and I am happy to finally release it.

Key changes in this update:

  • Improved Wayland surface management for desktop widgets
  • Better support for multi-monitor configurations
  • Updated compositor protocol to handle DPI scaling
  • Fixed several rendering issues with OpenGL/EGL contexts
  • Improved input handling for touch and pointer events

PlexyDesk is a desktop extension framework that provides compositor-level desktop widgets. The Wayland bridge allows PlexyDesk to integrate with Wayland compositors, providing a protocol for desktop shell integration.

The compositor protocol is designed to be compositor-agnostic, so it works with wlroots-based compositors, KWin, Mutter, and others. The goal is to provide a standard way for desktop extensions to communicate with the compositor without being tied to any specific implementation.
