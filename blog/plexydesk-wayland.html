PlexyDesk Wayland Bridge Update

2023-05-02

I have updated the PlexyDesk Wayland bridge to support the latest Wayland protocols and improve compatibility with modern compositors.

The Wayland bridge allows PlexyDesk widgets to render on Wayland sessions, which is increasingly important as distributions move away from X11. The main changes in this update:

  • Migrated to the xdg-shell stable protocol
  • Added support for layer-shell for desktop overlay widgets
  • Fixed rendering glitches on HiDPI displays
  • Improved input handling for touch devices

If you are using PlexyDesk on a Wayland session, please update to the latest build and report any issues on the GitHub repo.
